Question 1: Analogy (Simple Sequence)

Theory

Alphabet analogy problems require you to identify the relationship between the first pair of letter groups and apply the same relationship to the second pair to find the missing group.

Question and Options

BC : DE :: EF : ?

  • A) GH
  • B) FG
  • C) HI
  • D) IJ

Diagrams

Pattern Analysis for BC : DE B -> C (+1, consecutive) D -> E (+1, consecutive) Relationship between pairs: The second pair (DE) starts with the letter immediately following the last letter of the first pair (C -> D). Applying the pattern to EF : ? E -> F (+1, consecutive) The answer must start with the letter immediately following F, which is G. The answer must be a consecutive pair. Result: G -> H (+1)

Answer

A) GH

Explanation

  1. The relationship between B and C is that they are consecutive letters.
  2. Similarly, D and E are consecutive letters.
  3. The connection between the two pairs is that 'D' comes immediately after 'C'.
  4. Following this logic, the missing pair must come immediately after 'F'. The letter after 'F' is 'G'.
  5. Since the pair must be consecutive letters, the answer is GH.

Additional Remarks

This is the most basic type of alphabet analogy, focusing on direct succession. Always check for simple consecutive patterns first.

Question 2: Analogy (Consistent Gaps)

Theory

This type of analogy involves two layers of logic: the "internal pattern" (the gap between letters within a group) and the "external pattern" (the relationship between the groups themselves).

Question and Options

CEG : HJL :: IKM : ?

  • A) LNP
  • B) MOP
  • C) NPR
  • D) OQS

Diagrams

Internal Pattern Analysis: C (+2) -> E (+2) -> G (Skip one letter) H (+2) -> J (+2) -> L (Skip one letter) I (+2) -> K (+2) -> M (Skip one letter) External Pattern Analysis (Relationship between pairs): The second group (HJL) starts with H, which is the letter immediately after G from the first group. G -> H (+1) Applying to IKM : ? The answer must start with the letter after M, which is N. The answer must follow the internal (+2) pattern. N (+2) -> P (+2) -> R Result: NPR

Answer

C) NPR

Explanation

  1. Internal Logic: In CEG, each letter is separated by one skipped letter (C, skip D, E, skip F, G). This is a "+2" jump pattern. The same applies to HJL and IKM.
  2. External Logic: The first letter of the second group (H) is the very next letter after the last letter of the first group (G).
  3. Application: The last letter of IKM is M. The next letter is N. Therefore, our answer must start with N.
  4. Applying the internal "+2" pattern to N gives us: N, skip O, P, skip Q, R. The result is NPR.

Additional Remarks

Always dissect the problem into its internal and external logic. Solving them separately simplifies complex-looking questions.

Question 3: Analogy (Progressive Shift)

Theory

In this pattern, a different operation or "shift" is applied to each corresponding letter in the pair. The key is to find the pattern in the shifts themselves (e.g., +1, +2, +3 or +1, +3, +5).

Question and Options

CAT : DDY :: BIG : ?

  • A) CLL
  • B) CLM
  • C) CML
  • D) LMC

Diagrams

Shift Pattern Analysis for CAT -> DDY: C -> D (Shift of +1) A -> D (Shift of +3) T -> Y (Shift of +5) The pattern of the shifts is: +1, +3, +5 (consecutive odd numbers). Applying this shift pattern to BIG: B (+1) -> C I (+3) -> L (I is 9th letter, 9+3=12, L is 12th) G (+5) -> L (G is 7th letter, 7+5=12, L is 12th) Result: CLL

Answer

A) CLL

Explanation

  1. Compare the first letters: C becomes D, which is a shift of +1 position.
  2. Compare the second letters: A becomes D, which is a shift of +3 positions (A->B->C->D).
  3. Compare the third letters: T becomes Y, which is a shift of +5 positions (T->U->V->W->X->Y).
  4. The pattern of shifts is +1, +3, +5. We apply this same pattern to the word BIG.
  5. B + 1 = C.
  6. I + 3 = L.
  7. G + 5 = L.
  8. The resulting word is CLL.

Additional Remarks

When you see unrelated shifts, check if the shifts themselves form a logical sequence. This is a common advanced pattern.

Question 4: Odd One Out (Simple Gap)

Theory

In "Odd One Out" problems, you must find a common property or pattern shared by all but one of the options. The option that does not follow this pattern is the answer.

Question and Options

Find the odd one out from the following:

  • A) DE
  • B) PQ
  • C) TU
  • D) MO

Diagrams

Analyzing the gap within each pair: A) D -> E (+1 jump, consecutive) B) P -> Q (+1 jump, consecutive) C) T -> U (+1 jump, consecutive) D) M -> O (+2 jump, skips letter N) The common pattern is a +1 jump. MO does not fit.

Answer

D) MO

Explanation

  1. In option A, D and E are consecutive letters in the alphabet.
  2. In option B, P and Q are consecutive letters.
  3. In option C, T and U are consecutive letters.
  4. In option D, M and O are not consecutive; the letter N is skipped between them.
  5. Therefore, MO is the odd one out as it is the only pair that is not consecutive.

Additional Remarks

The simplest property to check for in pairs is consecutiveness or a consistent gap. Start there before looking for more complex patterns.

Question 5: Odd One Out (Positional Pairs)

Theory

This problem involves recognizing patterns that use both the forward alphabet (A=1, B=2...) and the reverse alphabet (Z=1, Y=2...). A common pattern is pairing letters that are equidistant from opposite ends of the alphabet (e.g., A and Z, B and Y).

Question and Options

Find the odd one out from the following:

  • A) ABZ
  • B) CDX
  • C) EFW
  • D) GHT

Diagrams

Analyzing the pattern in each group: A) ABZ: A is 1st from start. B is 2nd. Z is 1st from end. Pattern: First and last letters are positional opposites (A-Z). B) CDX: C is 3rd from start. D is 4th. X is 3rd from end. Pattern: First and last letters are positional opposites (C-X). D) GHT: G is 7th from start. H is 8th. T is 7th from end. Pattern: First and last letters are positional opposites (G-T). C) EFW: E is 5th from start. F is 6th. W is 4th from end. The 5th letter from the end is V, not W. This breaks the "positional opposite" pattern.

Answer

C) EFW

Explanation

  1. The pattern in the majority of options is that the first and third letters are a pair from opposite ends of the alphabet (A is 1st, Z is last; C is 3rd from start, X is 3rd from end; G is 7th from start, T is 7th from end).
  2. Additionally, the first two letters are always consecutive (AB, CD, GH).
  3. Let's check option C: E and F are consecutive, which fits the first part of the pattern. However, E is the 5th letter from the start. The 5th letter from the end is V. The group given is EFW, not EFV.
  4. Since EFW does not follow the positional pair rule, it is the odd one out.

Additional Remarks

Familiarity with reverse pairs (A-Z, B-Y, C-X, etc.) is extremely helpful for solving these types of problems quickly. They are a very common pattern in reasoning tests.
