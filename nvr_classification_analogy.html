Classification Analogy

What is Classification Analogy?

Classification Analogy questions present you with a set of figures that share a common characteristic or belong to a group. Your task is to analyze this set, determine the underlying rule that unifies them, and then select the one figure from the answer options that also fits this rule.

How to Solve Classification Problems

  1. Analyze the Set Figures: Look at the group of figures provided (usually 3 or 4). Ignore the answer choices for now. The goal is to find a single, consistent property that every figure in this set shares.
  2. Formulate the Common Rule: What do they all have in common? Systematically check for properties like:
    • Number of elements: Do they all have the same number of sides, lines, or dots?
    • Shape Type: Are they all made of straight lines? Or curved lines? Are they all polygons?
    • Symmetry: Do they all possess a line of symmetry (vertical, horizontal)?
    • Shading: Is there a consistent pattern of filled vs. unfilled elements?
    • Position: Is a specific element always in a certain position (e.g., a dot in a corner)?
    • Relationship: Do the elements within each figure share a relationship (e.g., the inner shape has fewer sides than the outer)?
  3. Test the Answer Options: Once you have a clear rule, apply it to each of the answer choices. Only one of them will satisfy the rule you've identified.
