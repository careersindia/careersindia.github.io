Calendar Tricks

Core Concept: The Odd Day Method and Codes

Question

How can you find the day of the week for any date without a calendar?

Answer

By using a formula that combines five key values: the date, month code, year, number of leap years, and century code. The final sum is divided by 7, and the remainder gives the day of the week.

If you know these codes, any problem becomes a simple 30-second calculation."

The Five-Step Formula

To find the day for any date (e.g., 26th January 1950), you sum five values:

  1. Date: The day of the month (e.g., 26).
  2. Month Code: The code for the given month from the table (e.g., January = 0).
  3. Year: The last two digits of the year (e.g., for 1950, it's 50).
  4. Leap Years: The number of leap years within that year part. Calculated as (Last two digits of year) / 4, ignoring the remainder (e.g., 50 / 4 = 12).
  5. Century Code: The code for the century from the table (e.g., for the 1900s, it's 0).

Finally, Total Sum ÷ 7, and the remainder is your day code.

Important Note on Leap Years: "For January or February of a leap year, you must subtract 1 from the final sum *before* dividing by 7. This is a special rule that only applies to those two months in a leap year. For any other month, or for any month in a normal year, you don't need to do anything."

Example 1: Finding the Day for a Historical Date

Question

What was the day of the week on 26th January 1950?

Answer

Thursday.

Explanation

Let's apply the five-step formula:

  1. Date: 26
  2. Month Code (Jan): 0
  3. Year: 50
  4. Leap Years (50/4): 12
  5. Century Code (1900s): 0

Total Sum = 26 + 0 + 50 + 12 + 0 = 88.

Now, find the odd days by dividing by 7:

88 ÷ 7 gives a remainder of 4 (since 7 × 12 = 84, and 88 - 84 = 4).

The remainder is 4. Looking at the Day Codes table, a code of 4 corresponds to Thursday.

Example 2: A Date in the 21st Century

Question

What was the day of the week on 2nd October 1869 (Gandhi's Birthday)?

Answer

Saturday.

Explanation

Applying the five-step formula:

  1. Date: 2
  2. Month Code (Oct): 0
  3. Year: 69
  4. Leap Years (69/4): 17
  5. Century Code (1800s): 2

Total Sum = 2 + 0 + 69 + 17 + 2 = 90.

Find the odd days by dividing by 7:

90 ÷ 7 gives a remainder of 6 (since 7 × 12 = 84, and 90 - 84 = 6).

The remainder is 6. The Day Code for 6 is Saturday.
