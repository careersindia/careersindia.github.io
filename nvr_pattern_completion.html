Pattern Completion

What is Pattern Completion?

Pattern Completion problems present a 3x3 matrix of figures where one cell is empty. The figures in the matrix follow a specific rule or pattern, either horizontally (across rows) or vertically (down columns). Your goal is to decipher this pattern and select the answer choice that correctly fills the empty cell.

How to Solve Pattern Completion Puzzles

  1. Analyze Horizontally (Row Logic): Examine the figures in the first and second rows. How does the first figure change to become the second, and then the third? The same logical rule should apply to all three rows.
  2. Analyze Vertically (Column Logic): If no clear row pattern emerges, check the columns. How does the top figure change to become the middle one, and then the bottom one? The same rule should apply to all three columns.
  3. Identify the Core Pattern: The pattern can be based on several types of transformations:
    • Rotation: A figure rotates by a fixed angle (e.g., 45° or 90°) in each step.
    • Superimposition: The third figure in a row/column is a combination of the first two.
    • Subtraction: The third figure is what remains after removing common elements from the first two.
    • Progression: The number of elements (lines, dots, sides) increases or decreases in a logical sequence.
    • Movement: An element moves to a new position in each step.
    • Shading/Reflection: Elements change shading or are reflected.
  4. Apply the Rule to Find the Missing Figure: Once you've identified the rule, apply it to the row or column containing the missing figure to determine what it should look like.
