CODING – DECODING

Theory: Introduction

Coding is the way of encrypting the messages that are to be transmitted.

Decoding is the process of decrypting the messages.

These types of questions are asked to test the logical reasoning skills of a person. There are nearly five types in which the Coding Decoding questions can be framed.

  • Letter to Letter Coding: When alphabets in a word are replaced with some different alphabets by using a certain form of code or pattern.
  • Number to Letter Coding: In a word, some numbers get replaced with the alphabets as per a particular rule.
  • Substitutional Coding: Here the words are encrypted with a special code or word.
  • Chinese Coding: Here the statements will contain similar words as codes but in a different order.
  • Symbol Coding: In this type of coding decoding questions, the words will be coded with symbols, numbers, letter and operational signs.

Tips and Tricks to Solve Coding Decoding Questions

  • Read the question carefully and note down all the important words along with their given codes.
  • Try to observe the pattern of the code that is given to you. You have to use the hit and trial method and check for various rules.
  • Use the elimination method to simplify the question and remove the unnecessary values.

TYPE 1 - LETTER CODING

This is the code that is derived on the basis of alphabets. Usually, the position of alphabets is used.

Question: Example 1

In some language, “RESULT” is coded as “SFTVMU”. In the same language, how will we code “EXAM”?

  1. DWBL
  2. FZNB
  3. FYBN
  4. DYZN
Explanation

On observing the given code, we find the S follows R, F follows E, T follows S, V follows U, M follows L and U follows T in the English alphabet. Thus, we see that in this code the alphabet is shifted one step forward. Thus, the code for E will be F, X will be Y, A will be B and M will be N respectively.

Hence the correct option here is (c) FYBN.

Question: Example 2

If in a certain code, GLAMOUR is written as IJCNMWP and MISRULE is written as OGUSSNC, then how will TOPICAL be written in that code? (RBI 2005)

  1. VMRJECN
  2. VMRHACJ
  3. VMRJACJ
  4. VNRJABJ
  5. None of these
Page 38 of 309
Explanation: Solution to Example 2

It is observed that the first, third and sixth letters of the word have been moved two letters forward (+2). Also, the second, fifth and seventh letters have been moved two alphabets backward (-2) while the middle alphabet i.e., the fourth alphabet has been moved one step forward (+1) to get the corresponding codes of GLAMOUR and MISRULE.

Pattern: +2, -2, +2, +1, -2, +2, -2

Following the same pattern, the code for TOPICAL shall be VMRJACJ.

Hence, the correct answer is (c) VMRJACJ.

Question: Example 2

If each of the letters in the English alphabet is assigned an even numerical value by giving A=2, B=4 and so on, then what would be the total value of the letters for the word 'LADY' when similarly coded? (SSC-CGL 2013)

  1. 74
  2. 72
  3. 84
  4. 82
Explanation

The value is the alphabet's position multiplied by 2.

  • L is the 12th alphabet, so 12 x 2 = 24
  • A is the 1st alphabet, so 1 x 2 = 2
  • D is the 4th alphabet, so 4 x 2 = 8
  • Y is the 25th alphabet, so 25 x 2 = 50

Adding 24 + 2 + 8 + 50 we get 84.

Hence, the correct answer is (c) 84.

The code is the sum of the alphabetical positions of the letters.

  • BAG = B(2) + A(1) + G(7) = 10
  • BOX = B(2) + O(15) + X(24) = 41

Hence, the correct option is (c) 41.

ALPHABET TEST

INTRODUCTION

Alphabet test involves solving problems based on letters of English alphabet.

Some Important points to Remember:

  1. Vowels – The letters A, E, I, O and U are called vowels.
  2. Consonants – The letters other than vowels are called consonants.
  3. Letters A to M are called the first half of English alphabet.
  4. Letters N to Z are called the second half of English alphabet.
  5. nth letter to the left of mth letter from right end = (m+n)th letter from the right end.
  6. nth letter to the right of mth letter from left end = (m+n)th letter from the left end.

Alphabetical arrangement means arranging the words in the sequence in which they occur in the dictionary.
